# Incluir-aluno-3DAW-
código em php para incluir dados de alunos (nome, matrícula e etc.)
Durante o desenvolvimento deste sistema CRUD (Create, Read, Update, Delete), optei por gerenciar o estado da aplicação utilizando as Sessões nativas do PHP ao invés de gravar os dados em um arquivo de texto plano (.txt).
Vale ressaltar que apenas o estilo (CSS) foi feito com inteligência artificial, para melhorar a vizualizção do código em funcionamento.
